1. Pasos para subir un proyecto a netbeans
   - Selecciona el proyecto a subir
   - Con Netbeans:
     1. Crea un Add con click derecho en el proyecto -> Git -> Add
     2. Luego con la misma ruta anterior, seleccionar Commit
     3. Tras esto, en el menu de netBeans, selecciona Team->Remote->Push
     4. Pon tu URL de repositorio, usuario y token
     5. las ramas en la que va a estar el proyecto y le das a finalizar
   - Tras seguir lo anterior, ve a Github y comprueba si se subio el proyecto
